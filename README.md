## DDoS Attack Detection Using Random Forest

A machine learning solution to identify and classify Distributed Denial-of-Service (DDoS) attacks from network traffic data using the Random Forest algorithm.


## Overview
This project leverages the Random Forest classifier to detect DDoS attacks in network traffic. By learning patterns from labeled traffic records, the model accurately distinguishes between benign and malicious activity — enabling early-stage threat detection and network protection.

## Repository Structure

## Dataset
The model trains and evaluates on ddos1.csv, a labeled dataset of network traffic records.
Features used:
FeatureDescriptionPacket RateNumber of packets per secondInter-arrival TimeTime gap between consecutive packetsPacket SizeSize of individual packetsDurationLength of the network sessionSource / Destination PortPort numbers on both endsPackets & Bytes TransferredVolume of data exchangedProtocol TypeNetwork protocol (one-hot encoded)
Target column: label — either benign or ddos

## Getting Started

Clone the repository  

bash   git clone https://github.com/your-username/DDoS-detection-model.git  
   cd DDoS-detection-model  

Install dependencies  

bash   pip install -r requirements.txt  

Add the dataset  
Place ddos1.csv in the root project directory.  
Run the notebook  

bash   jupyter notebook ddos1.ipynb
