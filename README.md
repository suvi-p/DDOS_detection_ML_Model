## Overview

This project leverages the **Random Forest** classifier to detect DDoS attacks in network traffic. By learning patterns from labeled traffic records, the model accurately distinguishes between **benign** and **malicious** activity — enabling early-stage threat detection and network protection.

## Dataset

The model is trained and evaluated on `ddos1.csv`, a labeled dataset of network traffic records.

**Features used:**

| Feature | Description |
| --- | --- |
| Packet Rate | Number of packets per second |
| Inter-arrival Time | Time gap between consecutive packets |
| Packet Size | Size of individual packets |
| Duration | Length of the network session |
| Source / Destination Port | Port numbers on both ends |
| Packets & Bytes Transferred | Volume of data exchanged |
| Protocol Type | Network protocol (one-hot encoded) |

**Target column:** `label` — either `benign` or `ddos`

## Getting Started

**1. Clone the repository**

```bash
git clone https://github.com/your-username/DDoS-detection-model.git
cd DDoS-detection-model
```

**2. Install dependencies**

```bash
pip install -r requirements.txt
```

**3. Add the dataset**

Place `ddos1.csv` in the root project directory.

**4. Run the notebook**

```bash
jupyter notebook ddos1.ipynb
```
